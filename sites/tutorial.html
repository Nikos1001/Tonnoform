Tutorial
Track page:

Create Pattern: click the '+' button in the bottom right corner
Delete selected Pattern: click the '-' button in the bottom right corner
Scroll pattern list: w / s

Add note: click + drag
Delete note: right click a note
Scroll midi editor: up / down arrow keys

Add selected pattern to sequencer: click on sequencer
Remove pattern from sequencer: right click on pattern in sequencer
Switch selected instrument: click on bottom left corner
Pause / play sequencer: space
Scroll sequencer: left / right arrow keys


Instrument Page:

Create new Instrument: click the '+' button in the bottom right corner
Delete selected Instrument: click the '-' button in the bottom right corner
Scroll instrument list: w / s

Edit envelope: click + drag rectangles


Misc

Control S: save(NOTE: to use this, save project in the project tab first)
